# Сайт-визитка «Родион – Программист»

Статический одностраничный лендинг (HTML + CSS + JS, без сборки).

## Структура

- `index.html` — главная страница
- `404.html` — страница ошибки (GitHub Pages подхватывает автоматически)
- `css/style.css` — стили (цвета в oklch, адаптив: до 640px / 641–1024px / от 1025px)
- `js/main.js` — появление блоков при скролле (IntersectionObserver, уважает prefers-reduced-motion)
- `fonts/` — self-hosted Manrope (вариативный 400–800, latin + cyrillic)
- `favicon.svg`, `og.png` — иконка и превью для соцсетей
